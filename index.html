O que é o KBase?

O KBase foi criado com o intuito de ajudar os iniciantes na carreira de TI com informações úteis no dia-a-dia.

Assim, auxiliando em tarefas/problemas diários e otimizando tempo de buscas em sites, forums, posts e etc.

Boas práticas na area de TI:

Existem diversas práticas que podem ser aplicadas no seu cotidiano e podem fazer total diferença.
Listei algumas que eu considero as mais importantes (até o momento):

• OUÇA/LEIA O QUE FOI PEDIDO COM ATENÇÃO!
Resolver um problema é eficaz. Tratar para que não ocorra mais e o que faz a diferença de verdade.

• ENTENDER O QUE O CLIENTE "QUER" E O QUE ELE "PRECISA" É A CHAVE!
Tenha empatia para entender que ninguem nasce sabendo de tudo! O mesmo conhecimento que voce tem, o outro pode não ter.

• EVITE REALIZAR ALTERAÇÕES QUE NÃO PODEM SER REVERTIDAS!
Excluir, alterar ou mover um arquivo, registro, programa ou qualquer informação. Pode causar incidentes (sendo de baixa ou alta gravidade).
Sempre se certifique que está no caminho correto e que a ação é realmente necessaria.

• UMA COMUNICAÇÃO CLARA PODE RESOLVER A MAIOR PARTE DO PROBLEMA!
Explique de forma simples. Sem termos muito ténicos, sem pressa e de forma clara.

• BUSQUE SEMPRE APRIMORAR OS SEUS CONHECIMENTOS!
O que pode te destacar dos demais não é o quanto voce conhece, mas como voce aplica o seu conhecimento.

• SEMPRE USE FERRAMENTAS QUE SÃO CONFIAVEIS!.
Fuja de gabiarras, programas crackeados, ativações não licenciadas e afins. São "soluções" que podem causar diversos problemas.

• NA DUVIDA? PEÇA AJUDA!
Quando se sentir inseguro, não conhecer um procedimento ou tiver qualquer duvida. Peça ajuda.

• FOCO NO PROCEDIMENTO!
Ocorrerá casos em que voce saberá como resolver um problema e não terá "ferramentas" para resolver. Os procedimentos servem justamente para que esses casos sejam apurados/analisados. Onde é encontrada a melhor solução para como resolver. Não dê um passo maior que a sua perna! Peça ajuda nesses casos.

Comandos mais usados no CMD:

Para acessar o terminal, acesse o Menu Iniciar, digite “cmd” (sem as aspas) e tecle ENTER. Se a sua versão do Windows for superior ao Windows 8, basta teclar Windows + R, digitar “cmd” (sem aspas) e seguir com o mesmo processo. Note que sempre devemos acionar a opção “Executar como Administrador” para que todos os seus comandos funcionem.

IPCONFIG: O comando "ipconfig" exibe informações sobre a configuração de rede do seu computador.
Ele mostra o endereço IP, máscara de sub-rede, gateway padrão e outras informações relacionadas à conexão de rede.

PING: O comando "ping" é usado para verificar a conectividade entre o seu computador e um endereço IP ou nome de domínio específico. Ele envia pacotes de dados para o destino e aguarda uma resposta, o que pode ser útil para diagnosticar problemas de rede ou verificar se um servidor está respondendo.

SYSTEMINFO: O comando "systeminfo" exibe informações detalhadas sobre o sistema operacional e o hardware do seu computador. Ele fornece detalhes como o nome do sistema operacional, versão, fabricante do sistema, informações de hardware (como processador, memória RAM, etc.) e até mesmo informações sobre a configuração da rede.

SFC /SCANNOW: O comando "sfc /scannow" é usado para verificar e reparar arquivos do sistema corrompidos no Windows. Ele verifica a integridade dos arquivos protegidos do sistema e tenta substituir os arquivos corrompidos por versões saudáveis. Esse comando é útil quando ocorrem erros ou falhas do sistema relacionados a arquivos corrompidos.

CHKDSK: O comando "chkdsk" é usado para verificar e corrigir erros no disco rígido ou em uma unidade de armazenamento, verificando a integridade do sistema de arquivos e tentando reparar setores defeituosos ou problemas de alocação de arquivos.

REGEDIT: O comando "regedit" é usado para acessar o Editor de Registro do Windows - uma ferramenta avançada que permite visualizar e editar as configurações do registro do sistema operacional. É importante ter cuidado ao fazer alterações no registro, pois modificações incorretas podem causar problemas no sistema.
